# Trae 智能采集助手 (Trae Smart Collector) 安装指南

恭喜！您已经拥有了这套强大的“浏览器 -> Trae”无缝采集系统。
本系统包含两部分：**Python 后端服务** 和 **Chrome 浏览器插件**。

---

## 第一步：准备 Python 后端

1.  **打开终端**：在 Trae 中打开一个新的终端。
2.  **安装依赖**：
    运行以下命令，安装所需的 Python 库（支持 FastAPI, 网页抓取, 视频字幕提取, AI 调用）。
    ```bash
    pip install -r requirements.txt
    ```
3.  **启动服务**：
    运行以下命令启动后端“大脑”。
    ```bash
    python server.py
    ```
    *成功标志：看到 `Server starting on http://localhost:18000`*

---

## 第二步：安装 Chrome 插件

1.  打开 Chrome 或 Edge 浏览器。
2.  在地址栏输入 `chrome://extensions` (Chrome) 或 `edge://extensions` (Edge)。
3.  **开启“开发者模式”** (Developer mode)：通常在右上角。
4.  点击 **“加载已解压的扩展程序”** (Load unpacked)。
5.  选择本项目的 `trae-extension` 文件夹：
    *   路径：`e:\Working overseas\AI tools\Access Trae\trae-extension`
6.  安装成功后，您会在浏览器工具栏看到 Trae 的图标。

---

## 第三步：使用指南

### 1. 配置插件
*   点击浏览器右上角的 **Trae 图标**。
*   **Mode (模式)**：
    *   `Raw Extraction` (推荐)：仅提取网页/视频正文，速度快，免费。
    *   `AI Blog Rewrite`：调用 AI 将内容重写为博客文章（需填 API Key）。
*   **Save Path (保存路径)**：
    *   默认为 `materials` 文件夹。您可以修改为任意绝对路径，例如 `e:\Working overseas\MyNotes`。
*   **API Key**：
    *   如果您选择 AI 模式，请在此填入 OpenAI / DeepSeek 的 Key。

### 2. 开始采集
*   **网页采集**：在任意文章页面 **右键 -> Save to Trae**。
*   **视频采集**：在 YouTube 视频页面 **右键 -> Save to Trae**（会自动抓取字幕！）。
*   **链接采集**：在不想打开的链接上 **右键 -> Save to Trae**。

### 3. 查看结果
*   几秒钟后，您指定的文件夹中会自动生成一个 Markdown (`.md`) 文件。
*   如果是 AI 模式，它已经是一篇排版精美的技术博客了！

---

## 常见问题

*   **Q: 点击右键后提示 Error?**
    *   A: 请检查 `python server.py` 是否正在运行，且没有报错。
*   **Q: 视频无法提取字幕？**
    *   A: 确保该视频本身有 CC 字幕。如果完全没有字幕，系统只会保存标题和链接。
*   **Q: DeepSeek 怎么用？**
    *   A: 在 Popup 里选择 DeepSeek 模型，填入 Key 即可。无需修改 Base URL（后端已内置）。
