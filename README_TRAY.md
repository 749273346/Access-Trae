# Trae 智能助手 - 托盘启动器指南

本版本新增了系统托盘启动器，允许您在不打开编辑器的情况下独立运行 Trae 智能助手。

## 1. 安装依赖

请确保已安装最新依赖：

```bash
pip install -r requirements.txt
```

## 2. 启动应用

直接运行启动脚本：

```bash
python launcher.py
```

或者，如果您希望完全隐藏命令行窗口，请将 `launcher.py` 重命名为 `launcher.pyw` 并双击运行。

## 3. 功能说明

程序启动后，会在 Windows 任务栏右下角显示一个蓝色图标。

### 🖱️ 托盘菜单功能：

- **状态显示**: 顶部显示当前服务状态 (🟢 Running / 🔴 Stopped)。
- **🌐 Open Dashboard**: 在浏览器中打开主界面。
- **📋 Clip from Clipboard**: **(新功能)** 复制链接后点击此按钮，后台将直接采集该链接内容，无需打开界面。
- **📂 Open Library**: 打开素材保存文件夹。
- **🔄 Restart Services**: 重启后台服务。
- **📝 View Logs**: 查看运行日志，便于排查问题。
- **❌ Exit**: 退出程序。

## 4. 常见问题

- **端口占用**: 默认使用端口 `18000` (API) 和 `8501` (UI)。如果启动失败，请检查日志。
- **开机自启**: 如需开机自启，请为 `launcher.pyw` 创建快捷方式并放入 Windows `Startup` 文件夹。
