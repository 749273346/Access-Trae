# Trae 助手操作手册

## 1. 简介
Trae 助手是一个后台运行的系统托盘程序，它可以帮助您管理 Trae 的各项服务（Web 界面、API 服务等），而无需一直打开代码编辑器。

## 2. 如何启动
为了方便使用，我们为您准备了一个一键启动脚本。

1.  找到项目文件夹根目录下的 **`双击启动.bat`** 文件。
2.  **双击**该文件。
3.  会出现一个黑色的命令行窗口，提示“正在启动...”。
4.  稍等片刻，窗口会自动关闭（或提示您按任意键关闭）。
5.  此时程序已在后台运行。

## 3. 如何找到托盘图标
程序启动后，会在电脑桌面的 **右下角任务栏** 显示一个小图标。

*   **图标样式**：一个蓝色的正方形，中间有一个白色的 "T" 字母（或者类似的科技感图标）。
*   **注意**：如果任务栏图标太多，它可能会被隐藏在 **“^” (显示隐藏的图标)** 箭头里。请点击那个小箭头查找。

## 4. 功能介绍
**鼠标左键** 或 **右键** 点击托盘图标，会弹出一个菜单，包含以下功能：

### 🖥️ 状态与控制
*   **服务状态**：显示当前服务是“🟢 在线”还是“🔴 离线”。
*   **打开仪表盘**：在浏览器中打开 Streamlit Web 界面。
*   **重启服务**：如果遇到卡顿或错误，可以点击此项重启后台服务。

### ⚡ 快捷操作
*   **清空剪贴板**：一键清空系统剪贴板内容。
*   **测试 API**：发送一个测试请求，检查服务是否正常响应。

### ⚙️ 系统维护
*   **开机自启**：
    *   勾选后，下次开机时会自动启动 Trae 助手。
    *   再次点击可取消勾选。
*   **重新生成图标**：如果您不喜欢当前的图标，或者图标显示异常，可以点击此项重新生成并刷新图标。
*   **查看日志**：打开日志文件夹，查看运行记录，方便排查问题。

### ❌ 退出
*   点击“退出”将彻底关闭 Trae 助手及其所有后台服务。

## 5. 常见问题
**Q: 图标没有变成新的？**
A: Windows 系统有时候会缓存旧的托盘图标。
1.  尝试在托盘菜单中点击 **“⚙️ 系统维护 -> 🔄 重新生成图标”**。
2.  如果还是没变，尝试重启电脑，或者重启 Windows 资源管理器。
3.  确保您是通过 `双击启动.bat` 或 `launcher.py` 启动的，而不是旧的方式。

**Q: 启动后没反应？**
A: 请查看 `logs` 文件夹下的 `launcher.log` 文件，里面记录了程序的运行错误信息。
